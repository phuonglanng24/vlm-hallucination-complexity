# Scene Complexity, Grounding, and Object Hallucination in Vision-Language Models

Code for the paper: *"Scene Complexity, Grounding, and Object Hallucination in Vision-Language Models"*

## Overview

We investigate whether scene-level visual complexity drives object hallucination in vision-language models (VLMs). Using LLaVA-1.5-7B on MS COCO 2014, we find that the relationship between complexity and hallucination is weak and measure-dependent, while object-level grounding is a much stronger and more consistent predictor.

This repository contains code for:
- Generating captions with LLaVA-1.5-7B
- Evaluating object hallucination with CHAIR (whole-caption and sentence-level)
- Computing three visual complexity measures (handcrafted, visual-statistics, SAM-based)
- Training an interpretable hallucination detector (XGBoost, 8 features)
- Running the mediation analysis (complexity → grounding → hallucination)
- The SAM region-removal intervention (H2)
- Reproducing all figures

## Requirements

- Python 3.9+
- PyTorch 2.0+ with CUDA
- ~32 GB GPU memory for SAM ViT-H (smaller GPUs work for everything else)

## Data

**MS COCO 2014 validation set:**
- Images: [download from COCO](https://cocodataset.org/#download) (`val2014`, ~7 GB, 40,504 images)
- Annotations: `instances_val2014.json` and `captions_val2014.json` from the same page

**SAM checkpoint (ViT-H):**
- [Download from Meta](https://github.com/facebookresearch/segment-anything#model-checkpoints) (`sam_vit_h_4b8939.pth`, ~2.4 GB)

**Image subsets:**
- `subsets/eval_900_ids.csv` — the 900 images used for H1 and H2 (included in this repo)
- `subsets/subset_5000_ids.csv` — the 5,000 images used for mediation (generated by the pipeline)

## Reproducing the Results

### 1. Caption generation (baseline)

```bash
python captioning/caption_llava.py --images val2014 --out captions.jsonl
```

For OPERA, REVERSE, and Attention Lens, use their official implementations:
- [OPERA](https://github.com/shikiw/OPERA)
- [REVERSE](https://github.com/tsunghan-wu/REVERSE)
- [Attention Lens](https://github.com/Ziwei-Zheng/Attention-Lens)

All methods use the same prompt (`Describe this image in detail.`) and `max_new_tokens=128`.

### 2. CHAIR evaluation

```bash
# Prepare captions for CHAIR (convert image_id to integer)
python captioning/prep_for_chair.py captions.jsonl captions_for_chair.jsonl

# Run CHAIR
python chair/chair.py --cap_file captions_for_chair.jsonl --caption_key text \
    --coco_path ./annotations --save_path chair_output.json

# Convert to labels CSV
python captioning/chair_to_labels.py chair_output.json labels.csv
```

### 3. Sentence-level CHAIR

```bash
# Split captions into sentences
python chair/sentence_level_chair.py --mode split \
    --input captions_for_chair.jsonl --output sentence_captions.jsonl

# Run CHAIR on sentences
python chair/chair.py --cap_file sentence_captions.jsonl --caption_key text \
    --coco_path ./annotations --save_path sentence_chair_output.json

# Aggregate to per-image metrics
python chair/sentence_level_chair.py --mode aggregate \
    --input sentence_chair_output.json --output sentence_level_metrics.csv
```

### 4. Complexity scores

```bash
# Handcrafted (4 features)
python complexity/compute_handcrafted.py --images val2014 \
    --instances instances_val2014.json --ids subsets/eval_900_ids.csv \
    --score --output hand_features.csv

# Visual-statistics (11 features)
python complexity/compute_vis_metrics.py --images val2014 \
    --ids subsets/eval_900_ids.csv --score --output vis_metrics.csv

# SAM-based (3 granularities, requires GPU)
python complexity/compute_sam_counts.py --images val2014 \
    --ids subsets/eval_900_ids.csv --checkpoint sam_vit_h_4b8939.pth \
    --score --output sam_counts.csv
```

### 5. Hallucination detector (full validation set)

```bash
python detector/hallucination_detector.py \
    --images val2014 --json captions.jsonl --csv labels.csv
```

### 6. Mediation analysis (5,000-image subset)

```bash
python mediation/complexity_grounding_pipeline.py --stage freeze \
    --labels labels.csv --images val2014 --instances instances_val2014.json
python mediation/complexity_grounding_pipeline.py --stage hand
python mediation/complexity_grounding_pipeline.py --stage vis
python mediation/complexity_grounding_pipeline.py --stage sam \
    --sam-checkpoint sam_vit_h_4b8939.pth
python mediation/complexity_grounding_pipeline.py --stage ground
python mediation/complexity_grounding_pipeline.py --stage all_analysis
```

### 7. SAM intervention (H2)

```bash
# Remove small regions and inpaint (produces modified images)
python intervention/sam_intervention.py --images val2014 \
    --ids subsets/eval_900_ids.csv --checkpoint sam_vit_h_4b8939.pth \
    --threshold 0.0005 --output intervened_0005/

python intervention/sam_intervention.py --images val2014 \
    --ids subsets/eval_900_ids.csv --checkpoint sam_vit_h_4b8939.pth \
    --threshold 0.001 --output intervened_001/

# Then re-caption with LLaVA and re-evaluate with CHAIR on each set
```

### 8. Figures

```bash
python figures/plot_table2.py --output figures/
python figures/plot_regression.py --input sentence_level_all_metrics.csv --output figures/
python figures/plot_intervention.py --output figures/
```
